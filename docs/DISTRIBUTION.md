# Distribution und Beiträge

Der öffentliche Quellcode von Sprachhilfe liegt in
[`tamaga3131400/sprachhilfe-source`](https://github.com/tamaga3131400/sprachhilfe-source).
Signierte und notarisierte macOS-DMGs sowie der Sparkle-Appcast liegen in
[`tamaga3131400/sprachhilfe-dist`](https://github.com/tamaga3131400/sprachhilfe-dist).
Der Appcast verweist jeweils auf den passenden öffentlichen Source-Tag.

Dieses Repository ist ein versionierter Quellspiegel: Jeder öffentliche Tag
enthält einen geprüften Release-Snapshot.

## Lokaler Build

Für einen Entwicklungsbuild werden Xcode, macOS 14 oder neuer und die in den
beiden `Package.resolved`-Dateien festgehaltenen Swift-Paketversionen benötigt.
